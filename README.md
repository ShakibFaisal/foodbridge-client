# FoodBridge - Connect & Donate Extra Food

[Live Site URL](https://your-live-site-url.com)

FoodBridge is a community-driven platform that helps users donate and receive extra food safely and efficiently.

## Features

- **Add Food:** Users can easily post their extra food along with details like quantity, location, expiry date, and images.
- **Browse Available Food:** All users can browse available food in their area with visually appealing cards and animations.
- **Request Food:** On the food details page, users can request food directly from the donator.
- **Donator Actions:** Donators can view incoming requests and either accept or reject them. If accepted, the food status automatically updates to "Donated".
- **Interactive UI:** Built with React, Tailwind CSS, and DesiUI, enhanced with Framer Motion animations, React Icons, Toast notifications, and SweetAlert2 for a modern, interactive experience.
- **User Authentication:** Secures user actions and ensures only registered users can donate or request food.
- **Responsive Design:** Fully mobile-friendly and accessible across devices.

## Tech Stack

- React.js
- Tailwind CSS + DesiUI
- Node.js + Express.js
- MongoDB
- Framer Motion (animations)
- SweetAlert2 & React Hot Toast (notifications)
- React Icons
